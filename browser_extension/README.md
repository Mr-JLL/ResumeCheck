# 51job 简历抓取助手 · 浏览器扩展

Chrome 和 Edge **均支持**，使用完全相同的代码，无需修改。

## 安装方法（开发者模式，无需上架商城）

### Chrome
1. 打开 `chrome://extensions`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本目录（`browser_extension` 文件夹）

### Edge
1. 打开 `edge://extensions`
2. 左下角开启「开发人员模式」
3. 点击「加载解压缩的扩展」
4. 选择本目录（`browser_extension` 文件夹）

## 配置与使用

1. 安装后点击浏览器工具栏的扩展图标
2. 填入服务器地址（如 `http://192.168.1.100:5000`）和设备名称
3. 点击「保存并连接」，状态变为绿色表示成功
4. 在服务器驾驶舱的「抓取设备」列表中选中本设备
5. 点击「打开浏览器」→ 51job 标签页自动打开，手动登录
6. 回到驾驶舱点击「立即抓取」，扩展在本机完成操作

## 局域网分发（多台电脑安装）

将整个 `browser_extension` 文件夹复制到共享网络路径，
其他电脑按上述步骤加载同一文件夹即可。
